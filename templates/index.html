Sample Questions

Q1: How do I configure VLAN tagging on TJ1400?

A: Navigate to the web console > Network Settings > VLAN Configuration. Add a new VLAN ID and associate it with the respective port.

Q2: What is the max throughput supported by TJ1600 in ring topology?

A: The TJ1600 supports up to 100 Gbps per lambda in a DWDM ring. Total throughput depends on configuration and number of active channels.

Q3: How do I retrieve alarms using CLI?

A: Use the command show alarms all to fetch all active alarms from the device's management interface.
